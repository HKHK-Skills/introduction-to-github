## Tere tulemast

Inimesed kasutavad GitHubi maailma kõige arenenumate tehnoloogiate ehitamiseks. Olgu selleks andmete visualiseerimine või uue mängu loomine - GitHubis on terve kogukond ja tööriistad, mis aitavad sul seda veelgi paremini teha. GitHub Skills'i "Sissejuhatus GitHubi" kursus juhendab sind läbi kõige vajaliku, et alustada panustamist vähem kui tunniga.

- **Kellele see on**: Uutele arendajatele, uutele GitHubi kasutajatele ja õpilastele.
- **Mida õpid**: Tutvustame repositooriume, harusid, commit'e ja pull request'e.
- **Mida ehitad**: Loome lühikese Markdown-faili, mida saad kasutada oma [profiili README-na](https://docs.github.com/account-and-profile/setting-up-and-managing-your-github-profile/customizing-your-profile/managing-your-profile-readme).
- **Eeltingimused**: Puuduvad. See kursus on suurepärane sissejuhatus sinu esimeseks päevaks GitHubis.
- **Kestus**: Selle kursuse läbimine võtab vähem kui tund.

Sellel kursusel sa:

1. Lood haru (branch)
2. Teed commit'i
3. Avad pull request'i
4. Ühendad (merge) oma pull request'i

### Kuidas alustada seda kursust

[![start-course](https://user-images.githubusercontent.com/1221423/235727646-4a590299-ffe5-480d-8cd5-8194ea184546.svg)](https://github.com/new?template_owner=HKHK-Skills&template_name=introduction-to-github&owner=%40me&name=skills-introduction-to-github&description=Minu+kursuse+repo&visibility=public)

1. Tee paremklikk nupul **Start course** ja ava link uuel vahekaardil.
2. Uuel vahekaardil täituvad enamik välju automaatselt.
   - Omanikuks (owner) vali oma isiklik konto või organisatsioon.
   - Soovitame luua avaliku (public) repositooriumi, kuna privaatsed repod [kasutavad Actions minuteid](https://docs.github.com/en/billing/managing-billing-for-github-actions/about-billing-for-github-actions).
   - Keri alla ja kliki vormi allosas nuppu **Create repository**.
3. Pärast uue repositooriumi loomist oota umbes 20 sekundit, seejärel värskenda lehte. Järgi samm-sammult juhiseid uue repositooriumi README-s.
